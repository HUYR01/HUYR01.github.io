云计算技术实验三实验报告

发表于 2023-03-21

山东大学云计算技术实验报告

学号：202000130231 姓名： 胡彦蓉 班级：数据20
实验题目：利用云平台搭建个人博客
实验学时：2 实验日期： 2023.3.14
实验目的：熟悉个人博客系统的搭建。具体包括：参考方案：注册Github账号，搭建Hexo环境并实现个人博客搭建，撰写实验报告。
硬件环境： 联网的计算机一台
软件环境：Windows or Linux

实验步骤与内容：

安装node.js

安装hexo

新建一个空白文件夹用于存放 hexo 资源。初始化hexo

打开hexo服务。在浏览器输入localhost:4000就可以看到生成的博客（使用ctrl+c可以把服务关掉）

注册Github账户。点击New repository。创建一个和用户名相同的仓库，后面加.github.io。点击create repository。

回到git bash中（在myblog文件中右键选择git bash here），生成SSH添加到GitHub

找到生成的.ssh文件。在GitHub的setting中，找到SSH keys的设置选项，点击New SSH key把你的id_rsa.pub里面的信息复制进去。

测试是否成功

将本地hexo文件更新到GitHub库中

1）打开自己的项目HUYR01.github.io

2）复制ssh地址

3）打开站点配置文件 _config.yml，翻到最后，修改为

4）安装deploy-git

5）输入hexo g hexo d

至此博客已搭建完成，我的个人网站的地址是 HUYR01.github.io

博客发布

1）使用命令创建.md文件并且指定文章标题

$ hexo n “文章标题”

2）编写.md文件，存储在\myblog\source_posts写

执行命令将编写好的文章推送到服务器上就可以通过个人博客连接查看了

$ hexo g

$ hexo d

我的个人博客网址为胡彦蓉的博客 (huyr01.github.io)

部署博客后，任何人都可以通过互联网访问其他人的博客。

博客是以超级链接为入口的网络日记，它代表着新的生活、工作和学习方式。

在现在这个信息时代，每个人都可以创建自己的个人博客。我们可以在博客中记录自己的生活和学习。同时，博客也是一个与他人交流的方式，我们可以通过他人的博客学习新知识，也可以给他人的学习提供帮助。
